# Contributing to EnvVault

Thanks for your interest in improving EnvVault!

## Reporting Bugs
- Include OS and jq version
- Command that failed
- Expected vs actual behavior
- Never include actual secrets in bug reports!

## Suggesting Features
- Describe the use case
- Consider security implications
- Provide examples

## Pull Requests
1. Fork repository
2. Create feature branch
3. Test with dummy data (never real secrets!)
4. Follow code style
5. Update documentation

## Security
- Test with dummy/test data only
- Never commit real secrets
- Consider encryption and privacy

## Testing
All tests should use fake/test credentials only.
